# Vehicle Detection YOLOv4-tiny

## Deskripsi

Proyek ini merupakan sistem deteksi dan penghitungan kendaraan yang masuk ke Kapal Ro-Ro Bengkalis menggunakan algoritma YOLOv4-tiny.

## Teknologi

- Python
- OpenCV
- YOLOv4-tiny
- MySQL

## Fitur

- Deteksi kendaraan
- Penghitungan kendaraan
- Penyimpanan data ke MySQL
